All options — new SigmetDraw(options)

OptionTypeDefaultWhat it does
adapterMapAdapter—Required. Wraps your map — new MapLibreAdapter({ map }), new OpenLayersAdapter({ map }) or new LeafletAdapter({ map }) (from @softwarity/sigmet-draw/maplibre · /openlayers · /leaflet).
firFeature | Polygon | MultiPolygon—Required. FIR polygon — used for clipping, the centre-inside-FIR constraint and line-endpoint snapping.
label(r) => string—Dynamic text drawn on the shape (return "" to hide).
tooltip(r) => string—Floating text shown on hover over the geometry.
nauticalMilesOnlybooleanfalseForce radii/widths to NM (never emit KM).
readonlybooleanfalseStart frozen — no handles/toolbar, no editing (area + label stay). Toggle later via setReadonly.
toolbarboolean | ToolbarConfig—Render the turnkey native toolbar (see below). Omit for none.
styleSigmetStyleInputDEFAULT_STYLEPartial style override, merged onto the defaults.

TokenFieldsWhat it styles
